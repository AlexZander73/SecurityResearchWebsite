I Downloaded a Suspicious File

Stay calm first

Downloaded something sketchy? Contain first, then verify what executed and what changed.

What the risk depends on

Risk depends on file type, whether you opened it, and system privileges at execution time. A downloaded but unopened file is much lower risk than one that executed.

Immediate steps

  1. Disconnect the device from the network if you opened the file.
  2. Run trusted security scans and quarantine detections.
  3. Remove unknown startup items or recently installed applications.
  4. Change important passwords from a separate clean device.

Priority order

  1. Contain the affected endpoint.
  2. Preserve evidence (file hash, timestamp, URL).
  3. Verify account compromise indicators.
  4. Rebuild device if integrity is uncertain.

Who to contact

For work devices, involve IT/security before self-remediation. For personal devices, consult vendor support if high-value accounts were exposed.

What to monitor

  • New background processes or extensions
  • Unusual CPU/network activity
  • Login notifications from unknown locations
  • Disabled security settings

Prevention for next time

Disable automatic execution of downloads, keep OS and browser patched, and avoid opening files from unexpected channels.

Important: Fast containment beats perfect certainty. Start with reversible safety steps now.
